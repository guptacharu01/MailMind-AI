**MailMind - AI**

MailMind-AI is a Smart Email Assistant that helps you **generate AI-powered email responses** using Spring Boot and React.  

## Features
-  AI-powered email generation
-  Smart subject line suggestions
-  User-friendly web interface (React)
-  RESTful backend built with Spring Boot
-  Integrated with Spring AI for natural language responses

##  Tech Stack
**Frontend:** React, HTML, CSS, JavaScript  
**Backend:** Java, Spring Boot, Spring AI  
**Version Control:** Git & GitHub  

##  Project Structure
MailMind-AI/

├── frontend/        # React frontend  
├── backend/         # Spring Boot backend  
└── README.md        # Project documentation  

##  How to Run the Project

###  Clone the Repository
```bash
git clone https://github.com/guptacharu01/MailMind-AI.git
cd MailMind-AI

cd backend
./mvnw spring-boot:run

cd frontend
npm install
npm start
```
